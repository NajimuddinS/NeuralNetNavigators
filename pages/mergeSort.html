Merge Sort Overview

Time Complexity

The time complexity of Merge Sort is O(n log n) in all cases (worst, average, and best).

Space Complexity

The space complexity of Merge Sort is O(n) due to the temporary arrays used for merging.

Applications

Merge Sort is used in various applications like:

  • Sorting linked lists.
  • External sorting algorithms.
  • Sorting large datasets that do not fit into memory.

Advantages

Merge Sort is stable and performs well on large data sets. It also divides the array into halves, making it easier to sort.

Disadvantages

Merge Sort requires additional space proportional to the size of the array, and it can be slower than other algorithms like Quick Sort for smaller arrays.
